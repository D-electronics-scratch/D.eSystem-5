# D.eSystem-5
D.eSystem 5 is the first OS from D.electronics which is written in C.
The features which are wirking are Max, News, Calculator and shutdown.

How to use the beta:
You can download the C file and open it in a C compiler like VS code,or if you have windows you can download the .exe and run it in your windows terminal. If you have no compile and windows you can go to this page: [▶ D.eSystem 5 im Online-Compiler starten](https://www.programiz.com/online-compiler/6HhztqxTrLKbp)
